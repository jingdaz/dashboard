# RBC Clear - Technical Assessment

## Problem Statement

Create a RESTful API with the ability to Add, Update and List the Federal Holidays for a country. There should also be an endpoint to upload a file which can add the holidays within it. For this assignment, please keep the country list to Canada and USA but implement it in a way so that the code is easy extensible. You can use any datastore you would like in memory or a SQL database but if it is a SQL database you should orchestrate it via docker and should be able to be run locally for the reviewer.

## Delivery Requirements

- The solution should be on Git and should run locally on a reviewer's machine.
- There should be a postman collection for your API provided with your delivery.
- There should be a swagger spec defined for your API also.
- It should be time-boxed in that the candidate must acknowledge receipt of the problem and they have 1 day to return with the solution.
- The solution should have more than 85% test coverage.
- The solution should practice clean code practices (no comment code, a README.md covering how to run locally, easy to follow code, and proper OOP practices).
- Given the time boxing please use a standard framework such SpringBoot or another framework that you will be able to walk the review through.
- Any assumptions made for the implementation should be mentioned in the README document along with any other points to note.
